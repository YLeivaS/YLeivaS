Calculadora de Consumo de Energía

Introduce una función de potencia P(t) en watts, el intervalo de tiempo, y calcula el consumo total de energía.

Ejemplos de funciones

Ejemplo 1:
Función con potencias:
pow(base, exponente)
Ejemplo 2:
Función con logaritmo:
log(t) para base euler
log10(t) para base 10
log2(t) para base 2
log(t, base) para otras bases
Ejemplo 3:
Función con raíz:
sqrt(t) para raiz cuadrada
nthRoot(t, indice) para otros indices
Notas:
Numeros especiales:
E para euler
pi para π
